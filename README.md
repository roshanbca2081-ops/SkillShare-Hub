# ShareSkill Hub

A premium mentorship and practical learning platform built with HTML5, CSS3, Bootstrap 5, Vanilla JavaScript, Core PHP, and MySQL.

## Features
- Modern landing page
- Auth pages for login and registration
- Dashboard and settings placeholders
- Responsive glassmorphism-inspired UI

## Run locally
1. Place the project in your XAMPP htdocs folder.
2. Start Apache and MySQL.
3. Open http://localhost/SkillShare%20Hub/
